# 个人壁纸空间

这是一个个性化的壁纸空间，集成了生日提醒、每日寄语和日程安排功能。

## 功能特点

- **自定义壁纸**：支持上传自己喜欢的壁纸作为背景
- **生日提醒**：添加和管理重要的生日日期
- **每日寄语**：记录对自己说的话和鼓励
- **日程安排**：管理每日任务和日程

## 使用方法

1. 直接在浏览器中打开 index.html 文件
2. 点击"上传新壁纸"按钮更换背景
3. 使用各个功能区添加和管理个人信息

## 注意事项

- 所有数据存储在浏览器的本地存储中，清除浏览器数据会导致信息丢失
- 默认壁纸需要放在 images/default-bg.jpg 位置
